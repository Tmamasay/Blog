关于一些学习笔记

instanceof

判断new出来的实例是不是属于某个构造函数的

包装属性

object.defineproperty(this,'属性名',{ get:function () { 对属性名要做的操作 }, set:function (value) { 对属性值做相关的限制 } });

关于定义that变量来保存this

在一个函数中的this指向不是当前的对象
`
function Product() {
this.name = {
name: ‘ss’,
age: ‘18’

};
var that = this

function aaa() {
    console.log(that.name.age);
}
aaa();

};
// aa();

Product.prototype = {
aa: function () {
console.log(this.name.name);
}
};
var ss = new Product();
ss.aa();
`

构造函数就是对象的模板

实例化就是把构造函数中的属性完全拷贝一分 在内存中新开辟一块地址来保存它；

检测数据类型

typeof–无法区分数组和json
toString.call(‘要检测的数据’) 能够区分数组和json
instanceof 判断new 出来的实例是不是属于某个构造函数的
constructor 判断某个实例他的构造函数是谁

定义构造函数是不会分配内存的 实例化才会分配内存

在后台拿到的是json字符串，要用JSON.parse(String)将字符串转换成json对象

构造函数就是普通函数组成的

当一个构造函数中出现了return并且return返回的是一个对象，那么就不会区使用构造函数中自定义的属性了 ，而去使用return返回的数据了
