面向对象=====万物皆对象

对象就是对工具进行分类的管理

变量

定义一个变量，在浏览器的解析中其实分两步，先声明变量，就是把声明的变量提到自己作用域的最前面
然后才给变量赋值。
` var a = 10;

(function test() {
    console.log(a);
    var a = 1;
    console.log(a);
})();
console.log(a);

//undefind 1 10 `

变量的搜索机制

在一个独立的作用域中，变量的搜索机制是先搜索局部变量，在
搜寻全局变量，还有变量的提升，var a=1；这种在浏览器解析时，会先把变量声明
提到前面，在进行赋值，就是两步。

函数

其实函数就是一个工具
封装性：我们只需要了解如何使用工具去实现某个方法，而不需要了解该工具是怎么生成的，这就是函数
的封装性

对象

工具=函数=方法

面向对象

就是借物的思想，学会的使用和管理工具。

面向对象的3个作用

1.面向对象编程–找到产品对象，分析方法，属性（单一职责原则）
2.数据描述–json–ajax交互
3.框架–解决浏览器兼容问题，封装一些常用的代码，就是多个对象
